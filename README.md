# MyProject

Developed with Unreal Engine 5





\# Projeto Unreal Engine 5



\## Objetivo do Projeto



Este projeto foi desenvolvido na Unreal Engine 5 com foco em criar um ambiente interativo em terceira pessoa.



O objetivo principal é explorar mecânicas básicas de movimentação, construção de cenário e interação inicial utilizando recursos da Unreal Engine.



\---



\## Funcionalidades atuais



\- Movimentação do personagem

\- Câmera em terceira pessoa

\- Ambiente explorável

\- Assets de vegetação e cenário

\- Estrutura inicial do gameplay



\---



\## Futuras implementações
